# APEX — BIST Analiz Terminali

**APEX**, Borsa İstanbul (BIST) yatırımcısına iki temel değer sunan bir analiz terminalidir: **risk disiplini** ve **bağlam**. Hisseye özgü volatilite hedefli pozisyon boyutlama ile gerçekleşen maksimum kayıpları bütçe sınırında tutar; aynı zamanda 90+ hissenin KAP açıklamalarını, haberlerini ve makro bağlantılarını anlık olarak izleyerek kullanıcının tek başına takip edemeyeceği gelişmeleri yüzeyine çıkarır. Amaç fiyat tahmini değil, gözden kaçan bilgiyi göstermek ve her işlemde risk disiplinini korumaktır.

**Yığın:** Python · Streamlit Community Cloud · GitHub Actions · yfinance  
**Strateji odağı:** Swing / pozisyonel (gün içi execution değil)
