# 🪒 Windows'ta Tek Tıkla Çalıştır

1. **Python kur (bir kez):** https://www.python.org/downloads/ → indir → kur.
   Kurulumda **"Add Python to PATH"** kutusunu işaretle (çok önemli).
2. Proje klasöründeki **`BASLAT.bat`** dosyasına **çift tıkla**.
   - İlk açılışta gerekli paketler otomatik kurulur (birkaç dakika).
   - Sonra uygulama tarayıcıda açılır.
3. Kapatmak için siyah pencereyi kapat.

> Excel rapor: uygulamada sol panelden **"📊 Excel rapor indir"**.
> Canlı veri gelmezse demo veriyle açılır (tüm ekranları/grafikleri görebilirsin).
